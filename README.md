# Real-Time Chat with Python

Simple real-time chat application built with Python, Flask and Socket.IO.

This project demonstrates how to implement real-time communication using WebSockets, focusing on clean code, project organization and backend fundamentals.

---

## Features

- Real-time message exchange
- Multiple users connected simultaneously
- WebSocket communication using Socket.IO
- Simple web interface

---

## Technologies

- Python 3.10+
- Flask
- Flask-SocketIO
- HTML / JavaScript
